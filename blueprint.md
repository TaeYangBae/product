# **Project Blueprint: Sensorium Web Re-creation**

## **Overview**
This project aims to recreate the website `sensorium.co.kr`, a professional site for **Sensorium**, the Korean agent for BEA (global sensor leader). The site focuses on advanced sensing technology for industrial, commercial, and safety applications.

## **Core Features**
1. **Hero Section**: High-impact visual representing "Peace of Mind" and "Future Sensing Technology."
2. **Solutions Showcase**: Categories like Smart Parking, Industrial Safety, People Counting, and Security.
3. **Product Gallery**: Overview of BEA's sensor products (EAGLE, FLY UP, etc.).
4. **Interactive Map/Contact**: Business location and contact details.
5. **Modern Navigation**: Sticky header with smooth scrolling.
6. **Responsive Design**: Mobile-first approach.

## **Visual Design Strategy**
- **Aesthetics**: Clean, corporate, and high-tech.
- **Color Palette**: 
  - Primary: Deep Blue (Trust) - `oklch(35% 0.15 250)`
  - Secondary: Tech Blue (Innovation) - `oklch(60% 0.2 250)`
  - Accent: Safety Orange/Red (Alert/Detection) - `oklch(65% 0.25 45)`
  - Background: Off-white with subtle texture.
- **Typography**: Modern Sans-serif (Inter or Pretendard for KR support).
- **Interactive Elements**: Hover effects, fade-in animations, and a Three.js background representing sensing fields.
